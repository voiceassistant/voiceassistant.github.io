Больше чем просто распознавание речи Помимо распознавания речи, API «Ассистента на русском» решает задачу распознавания смысла, установки и поддержания диалогового контекста, отображения интерфейса и синтеза речи для обратной связи с пользователем.

Все это теперь доступно разработчикам, кто не имеет знаний в технологиях ASR, TTS и NLP! API открывает вам возможность по внедрению голосового искусственного интеллекта в собственные приложения под Android.
